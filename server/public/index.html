Installation

Get FactionOps Running

Five minutes from zero to full war coordination.

1

Install a Userscript Manager

Get Tampermonkey for Chrome, Firefox, or Edge. Violentmonkey also works.

2

Install the FactionOps Script

Click the download button to install the userscript into Tampermonkey.

3

Open FactionOps Settings

Click the gear icon on any Torn war page to open configuration.

4

Enter Server URL & API Key

Your faction leader provides the server URL. Get your API key from Torn: Settings → API Keys.

5

Join a War

Open any ranked war page and FactionOps activates automatically.

Prerequisites

  • A userscript manager: Tampermonkey or Violentmonkey
  • A Torn API key (Settings → API Keys on torn.com)
